# ハンドレンジ
- SBで1レンジを使用するなら色付き以外コールから入る
- BBでは相手と同じ立場ならコールで入る
- 薄い色：レイズした相手の色にはコールで入る。
- 最緑：リレイズで入る.真に強いときはオールインまで乗る
