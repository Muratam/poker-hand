# ハンドレンジ
- BB: 相手と同じ立場ならコールで入る
- SB: BBを下ろすためにBTNにはリレイズで入る
- 最緑：リレイズで入る. AA,KK,QQ,AK ならオールインまで乗る(?)
- 薄い色：レイズにコールで入る。薄い色は相手のポジションが弱ければ。
- 10BB以下のオールイン：相手の席より大きいならオールイン。ほかはオリ
